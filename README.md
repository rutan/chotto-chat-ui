# chotto-chat-ui

> A simple Web UI for [Ollama](https://ollama.com/)

![](https://i.gyazo.com/40e1ca82b7996bd72eb5f8bc2b6787d8.jpg)

## How to use

### Web

#### Require

- node.js

#### Launch

```
$ corepack enable pnpm
$ pnpm install
$ cd apps/web
$ pnpm run dev
```

open http://localhost:5173
